# AlgVis

Algorithm Visualizer created with Qt and C++

Features include
- Sorting algorithms
  - Merge Sort
  - Quick Sort
  - Bubble Sort
  - Insertion Sort
  - Selection Sort
- Trees
  - BST
    - Insert
    - Remove
  - AVL Trees
    - Insert
    - Remove
- Graphs
  - BFS Traversal
  - DFS Traversal
  - Prim's MST Algorithm
  - Dijkstra's ALogrithm
- Maze Generation (Recursive Backtracker)
